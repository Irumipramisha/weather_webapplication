A sleek and responsive Weather Web Application built with **HTML, CSS, and JavaScript**. This app fetches real-time weather data from the **OpenWeatherMap API** to provide accurate information about any city worldwide.

## Features
- **Real-time Data:** Get up-to-date temperature, humidity, and wind speed.
- **Dynamic Search:** Search for weather details by entering any city name.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Interactive UI:** Clean, modern interface with smooth gradients.

## Technologies Used
- **HTML5:** For the structural layout.
- **CSS3:** For styling and professional UI design.
- **JavaScript (ES6):** For handling API calls and dynamic DOM updates.
- **OpenWeatherMap API:** To fetch live weather statistics.

## How to Use
1. Clone this repository or download the source code.
2. Open `index.html` in your web browser.
3. Enter a city name (e.g., Colombo, London) in the search bar.
4. Click the search icon or press **Enter** to see the results.

## Project Structure
- `index.html` - The core structure of the web app.
- `style.css` - Custom styling for a modern look.
- `script.js` - Logic for fetching and displaying API data.
